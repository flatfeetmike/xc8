12F1822 Support Information

#pragma config Usage

#pragma config <setting>=<named value>

For example:
// Clock Out Enable: CLKOUT function is disabled. I/O or oscillator function on the CLKOUT pin
// Watchdog Timer Enable: WDT controlled by the SWDTEN bit in the WDTCON register
// Power-up Timer Enable: PWRT disabled
// Flash Program Memory Code Protection: Program memory code protection is disabled
// Brown-out Reset Enable: Brown-out Reset enabled
// Fail-Safe Clock Monitor Enable: Fail-Safe Clock Monitor is enabled
// MCLR Pin Function Select: MCLR/VPP pin function is MCLR
// Data Memory Code Protection: Data memory code protection is disabled
// Internal/External Switchover: Internal/External Switchover mode is enabled
// Oscillator Selection: HS Oscillator, High-speed crystal/resonator connected between OSC1 and OSC2 pins
#pragma config CLKOUTEN = OFF, WDTE = SWDTEN, PWRTE = OFF, CP = OFF, BOREN = ON, FCMEN = ON, MCLRE = ON, CPD = OFF, IESO = ON, FOSC = HS

// Internal/External Switchover: Internal/External Switchover mode is enabled

#pragma config Settings

Register: CONFIG1 @ 0x8007

CLKOUTEN =Clock Out Enable
OFFCLKOUT function is disabled. I/O or oscillator function on the CLKOUT pin
ONCLKOUT function is enabled on the CLKOUT pin
WDTE =Watchdog Timer Enable
SWDTENWDT controlled by the SWDTEN bit in the WDTCON register
ONWDT enabled
OFFWDT disabled
NSLEEPWDT enabled while running and disabled in Sleep
PWRTE =Power-up Timer Enable
OFFPWRT disabled
ONPWRT enabled
CP =Flash Program Memory Code Protection
OFFProgram memory code protection is disabled
ONProgram memory code protection is enabled
BOREN =Brown-out Reset Enable
ONBrown-out Reset enabled
OFFBrown-out Reset disabled
NSLEEPBrown-out Reset enabled while running and disabled in Sleep
SBODENBrown-out Reset controlled by the SBOREN bit in the BORCON register
FCMEN =Fail-Safe Clock Monitor Enable
ONFail-Safe Clock Monitor is enabled
OFFFail-Safe Clock Monitor is disabled
MCLRE =MCLR Pin Function Select
ONMCLR/VPP pin function is MCLR
OFFMCLR/VPP pin function is digital input
CPD =Data Memory Code Protection
OFFData memory code protection is disabled
ONData memory code protection is enabled
IESO =Internal/External Switchover
ONInternal/External Switchover mode is enabled
OFFInternal/External Switchover mode is disabled
FOSC =Oscillator Selection
HSHS Oscillator, High-speed crystal/resonator connected between OSC1 and OSC2 pins
ECLECL, External Clock, Low Power Mode (0-0.5 MHz): device clock supplied to CLKIN pin
EXTRCEXTRC oscillator: External RC circuit connected to CLKIN pin
ECMECM, External Clock, Medium Power Mode (0.5-4 MHz): device clock supplied to CLKIN pin
LPLP Oscillator, Low-power crystal connected between OSC1 and OSC2 pins
INTOSCINTOSC oscillator: I/O function on CLKIN pin
XTXT Oscillator, Crystal/resonator connected between OSC1 and OSC2 pins
ECHECH, External Clock, High Power Mode (4-32 MHz): device clock supplied to CLKIN pin

Register: CONFIG2 @ 0x8008

STVREN =Stack Overflow/Underflow Reset Enable
ONStack Overflow or Underflow will cause a Reset
OFFStack Overflow or Underflow will not cause a Reset
BORV =Brown-out Reset Voltage Selection
LOBrown-out Reset Voltage (Vbor), low trip point selected.
HIBrown-out Reset Voltage (Vbor), high trip point selected.
LVP =Low-Voltage Programming Enable
ONLow-voltage programming enabled
OFFHigh-voltage on MCLR/VPP must be used for programming
WRT =Flash Memory Self-Write Protection
BOOT000h to 1FFh write protected, 200h to 7FFh may be modified by EECON control
HALF000h to 3FFh write protected, 400h to 7FFh may be modified by EECON control
OFFWrite protection off
ALL000h to 7FFh write protected, no addresses may be modified by EECON control
PLLEN =PLL Enable
ON4x PLL enabled
OFF4x PLL disabled
